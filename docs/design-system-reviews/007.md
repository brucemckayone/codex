## Key insight ‚Äî R12 mutation pattern

Iter-006 landed a fix for R12's half-step-spacing class. Iter-007 found R12 *hadn't been closed*; it had *mutated* to sparse fills. One token family got promoted, siblings stayed scoped, and the next agent writing `var(--space-9)` in system-scope stepped straight back into the same resolve-to-unset bug.

This is a general failure mode for bug classes that have a *taxonomy* (half-steps, sparse fills, brand colors, motion tokens). Fixing one genus of the taxonomy doesn't close the rule. The skill now names this explicitly so iter-008+ approaches token-scope promotion as "sweep the whole taxonomy" rather than "fix the one that's biting".
