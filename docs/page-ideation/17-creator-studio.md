# Creator Studio Pages — Feature Ideation

**Routes**: `creators.*/studio/*` — Dashboard, Content, Media, Settings
**Current state**: Mirrors org studio but for personal content. Basic functionality.
**Priority**: MEDIUM — personal dashboard for independent creators.

---

## Vision

The creator studio is the **personal workshop** — where creators manage their independent content, media library, and settings. Simpler than org studio (no team, no customers, no billing), but equally polished.

---

## Creator Studio Dashboard

### Simplified Version of Org Dashboard
- KPI strip: Revenue (personal only), Content count, Views, Downloads
- Revenue chart: Personal sales over time
- Recent sales: "Sarah purchased 'Morning Yoga' — £29"
- Quick actions: Create Content, Upload Media, Edit Profile, View Public Profile

### Cross-Org Summary
- "Your content across all organizations" summary
- Mini cards per org: Org logo + name + content count + revenue contribution
- "View in {Org} Studio" links
- Total combined revenue across all orgs + personal

### Onboarding for New Creators
- Checklist: Upload first media → Create first content → Set pricing → Publish
- Profile completion meter: "Your profile is 60% complete"
- Tips: "Add a bio", "Connect social links", "Upload an avatar"

---

## Creator Studio Content

### Same as Org Studio Content (doc #08), with:
- Personal content only (no org content visible)
- "Where to sell" option: Post to personal profile OR select an org
- Clear indication of where each content is published
- Revenue attribution: "Sold on personal profile" vs "Sold on {Org Name}"

---

## Creator Studio Media

### Same as Org Studio Media (doc #09), with:
- Personal media library (they own these forever)
- "Shared with" indicator: Which orgs have access to each file
- "Share with Org" action: Grant an org permission to use this media
- "Revoke from Org" action: Remove an org's access

---

## Creator Studio Settings

### Profile Settings
- Same as /account profile, but with creator-specific fields
- Creator tagline/title
- Teaching specialties/categories
- Featured content selection
- Background image/banner upload

### Payout Settings
- Stripe Connect account setup/status
- Payout history (personal sales)
- Revenue share settings per org (future)

---

## Priority Ideas (Top 5)

1. **Cross-org revenue summary** showing earnings across all orgs + personal
2. **Profile completion meter** guiding new creators to a complete profile
3. **"Share with org" controls** in media library for multi-org creators
4. **"Where to sell" selector** when creating content (personal vs org)
5. **Quick actions** for common creator workflows
