# Creator Content Pages — Feature Ideation

**Routes**:
- Content list: `creators.*/[username]/content`
- Content detail: `creators.*/[username]/content/[contentSlug]`
- Checkout success: `creators.*/checkout/success`
**Current state**: Basic content listing and detail pages. Similar patterns to org equivalents.
**Priority**: MEDIUM — creator's personal catalogue.

---

## Vision

The creator's content page is their **personal storefront** — a curated catalogue of everything they sell independently. It should feel personal and direct — "Buy from me" not "Buy from a platform."

---

## Content List Page

### Same as Org Explore, But Personal
- No need for "creator" filter (it's all from this creator)
- Categories become the primary filter
- Personal tone: "My Courses", "What I Teach"
- Option for creator to customize section headings

### Layout
- Category tabs: All | Yoga | Meditation | Nutrition
- Content grid with pricing
- Sort: Newest, Popular, Price
- "Free" and "Premium" filter toggles

### Personal Touch
- Creator's voice in the page intro: "Here's everything I've created..."
- Customizable section order (featured first, then by category)
- "Bundle Deals" section if bundles exist

---

## Content Detail Page

Same as org content detail (doc #05), with these differences:
- Creator's personal branding (not org branding)
- "More from {Name}" section shows personal content only
- No org badge — this is personal content
- "Also available on {Org}" badge if same content is on an org

---

## Checkout Success

Same patterns as org checkout (doc #06), with personal branding and tone.

---

## Priority Ideas (Top 3)

1. **Personalized intro** with creator's voice and customizable headings
2. **Category-first navigation** since all content is by one person
3. **Bundle/collection showcase** for curated content packages
