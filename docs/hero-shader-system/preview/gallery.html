Shader Hero Presets

Interactive shader backgrounds for org hero sections. Each preset is configurable with brand colors, mouse interaction, and animation controls.

Working Presets
Suture Fluid
Based on XddSRX by cornusammonis
Reaction-diffusion-advection system. Organic viscous fingering patterns that respond to mouse movement. Self-sustaining — patterns emerge and evolve continuously.
2-pass (ping-pong FBO) Fluid sim Approved
Ether
Based on MsjSW3 by nimitz
Raymarched 3D volumetric glow. Ethereal organic forms floating in branded space. Mouse creates parallax perspective shift. Scroll to zoom.
Single-pass Raymarch Delivered
Domain Warp
Technique from lsl3RH (domain warping)
Recursive FBM domain warping with bump-mapped lighting. Creates organic marble, cloud, and lava textures. Mouse magnifies warping near cursor.
Single-pass Procedural Verify fix
Water Ripple
Inspired by Mt33DH
2D wave equation with persistent ripples. Mouse hover creates continuous soft ripples, click for splashes. Normal-mapped surface with refraction and caustics.
2-pass (ping-pong FBO) Wave sim Building

Future
Architecture note: Each preset is a standalone HTML file with WebGL2. "Single-pass" presets run one fragment shader per frame (cheapest). "2-pass" presets use ping-pong framebuffers for persistent state (fluid sim, wave sim). All presets support brand color injection, mouse interaction, and configurable parameters. For production, these will be ported to Svelte components reading brand colors from the existing tokenOverrides system.
